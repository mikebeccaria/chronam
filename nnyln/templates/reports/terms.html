National Digital Newspaper Program Vocabulary

To support RDF views for resources in Chronicling America a few vocabulary elements needed to be created when suitable terms couldn't be found elsewhere. This page is not an exhaustive list of vocabulary terms in use in Chronicling America linked data views, but is simply a list of those classes and properties that have been created by the Library of Congress to support Chronicling America and the National Digital Newspaper Project more generally.

Term Name: Awardee
URI: http://chroniclingamerica.loc.gov/terms#Awardee
Label: Awardee
Definition: An institution that has produced digital newspaper content as part of the National Digital Newspaper Program.
Type of Term: Class
Refines: http://xmlns.com/foaf/0.1/Agent
Term Name: Batch
URI: http://chroniclingamerica.loc.gov/terms#Batch
Label: Batch
Definition: A set of files that represent number of newspaper pages and their metadata.
Type of Term: Class
Term Name: Page
URI: http://chroniclingamerica.loc.gov/terms#Page
Label: Page
Definition: A page that forms a part of an Issue.
Type of Term: Class
Term Name: Essay
URI: http://chroniclingamerica.loc.gov/terms#Essay
Label: Essay
Definition: A document written by an NDNP awardee that describes one or more NDNP Newspaper Titles
Type of Term: Class
Refines: http://xmlns.com/foaf/0.1/Document
Term Name: bag
URI: http://chroniclingamerica.loc.gov/terms#bag
Label: has Bag
Definition: Indicates when a batch is associated with a BagIt bag that is mounted on the web at a URL
Type of Term: Property
Term Name: sequence
URI: http://chroniclingamerica.loc.gov/terms#sequence
Label: Page Sequence
Definition: A sequence number uniquely identifying a newspaper page on a microfilm reel.
Type of Term: Property
Term Name: number
URI: http://chroniclingamerica.loc.gov/terms#number
Label: Page Number
Definition: A printed page number associated with a newspaper page.
Type of Term: Property
Term Name: section
URI: http://chroniclingamerica.loc.gov/terms#section
Label: Page Section
Definition: A label associated with the section of a newspaper issue that a newspaper page is in.
Type of Term: Property
